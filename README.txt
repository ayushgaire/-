【多言語システムについて】
-----------------------------------------------------------
・デフォルト言語は日本語です。
・画面右上（スマホは画面最上部）の「🇯🇵 日本語 / 🇺🇸 English」
  ボタンで、サイト全体の言語が即座に切り替わります。
・選んだ言語はブラウザに記憶され、次回訪問時も保持されます。
・翻訳テキストは js/i18n.js にまとめられています。
  文言を変更したい場合はこのファイルを編集してください。

-----------------------------------------------------------
【メニューの編集方法】★重要★
-----------------------------------------------------------
メニューの内容（名前・説明・価格・辛さ・画像）はすべて
  js/menu-data.js
で管理しています。このファイルを編集するだけで
メニューページが自動更新されます。

◆ 各メニュー項目の構成:
   ja        … 日本語名
   en        … 英語名
   descJa    … 日本語説明
   descEn    … 英語説明
   price     … 価格（数字のみ。例: 980）
   spice     … 辛さ 0〜5（0=辛さなし, 5=激辛）
   popular   … true で「人気」バッジ表示
   img       … 画像ファイル名（images/menu-items/ 内）

◆ メニュー写真の差し替え:
   images/menu-items/ フォルダ内に、menu-data.js の img で
   指定したファイル名（例: c-butter.jpg）の画像を置くと
   そのメニューの写真として表示されます。
   現在はすべて仮の画像が入っています。実際の料理写真に
   同じファイル名で上書きしてください。
   （推奨サイズ: 横600px × 縦440px 程度）

-----------------------------------------------------------
【画像のアップロード場所】
-----------------------------------------------------------
分かりやすいよう、専用フォルダを用意しています。
各フォルダ内に説明テキストが入っています。

◆ uploads/logo/ ...... お店のロゴ画像（logo.png）
◆ uploads/hero/ ...... トップのメイン写真（hero.jpg）
◆ uploads/dishes/ .... 人気メニューの料理写真（dish1〜6.jpg）
◆ uploads/gallery/ ... ギャラリー写真（gallery1〜8.jpg）
◆ images/menu-items/  メニュー各品の写真（上記参照）

画像を入れた後、同じファイル名で images/ 内の画像と
置き換えるとサイトに反映されます。

-----------------------------------------------------------
【ロゴについて】
-----------------------------------------------------------
現在、仮のロゴ（images/logo.png）を設置しています。
正式なロゴ画像ができましたら、uploads/logo/ に入れた後
images/logo.png と同じ名前で置き換えてください。
トップページのヒーロー部分に円形で表示されます。

【ファイル構成】
-----------------------------------------------------------
  index.html .......... トップページ
  menu.html ........... メニューページ
  404.html ............ エラーページ
  css/style.css ....... スタイルシート
